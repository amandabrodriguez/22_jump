## Description
This Unity 3D project is a simple endless runner where the player must jump over obstacles while continuously moving forward. It demonstrates core gameplay mechanics such as automatic forward movement, jump physics, obstacle spawning, collision detection, and basic game flow, developed as part of a learning course in Unity.

## Features
- Endless runner gameplay with automatic forward movement.
- Jump mechanic to avoid incoming obstacles.
- Obstacle spawning system with increasing difficulty over time.
- Basic collision detection and player failure handling.
- Simple character movement and jump physics.
- Clean and minimal scene setup focused on gameplay mechanics.
- Project developed as part of a Unity learning course focused on core gameplay foundations.

## Technical requirements:
- **Unity Version:** 6000.0.47f1 or higher.
- **Render Pipeline:** URP.
- **Platforms:** PC.
